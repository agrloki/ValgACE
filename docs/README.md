# Документация ValgACE

Добро пожаловать в документацию проекта ValgACE - драйвера для Anycubic Color Engine Pro под Klipper.

## Структура документации

### Основные документы

- **[README.md](../README.md)** - Главная страница проекта с обзором и быстрым стартом
- **[Установка](INSTALLATION.md)** - Подробное руководство по установке модуля
- **[Руководство пользователя](USER_GUIDE.md)** - Как использовать ValgACE
- **[Справочник команд](COMMANDS.md)** - Полный список всех команд G-code
- **[Конфигурация](CONFIGURATION.md)** - Настройка параметров модуля
- **[Решение проблем](TROUBLESHOOTING.md)** - Типичные проблемы и их решения

### Техническая документация

- **[Протокол](Protocol.md)** - Техническое описание протокола взаимодействия с ACE Pro (English)
- **[Протокол (русский)](Protocol_ru.md)** - Техническое описание протокола взаимодействия с ACE Pro

## Быстрая навигация

### Для новых пользователей

1. Начните с [README.md](../README.md) для общего обзора
2. Прочитайте [Установку](INSTALLATION.md)
3. Изучите [Руководство пользователя](USER_GUIDE.md)
4. Ознакомьтесь с [Справочником команд](COMMANDS.md)

### Для опытных пользователей

- [Конфигурация](CONFIGURATION.md) - тонкая настройка параметров
- [Протокол](Protocol.md) / [Протокол (русский)](Protocol_ru.md) - технические детали протокола
- [Решение проблем](TROUBLESHOOTING.md) - диагностика и решение проблем

### Для разработчиков

- Исходный код: `extras/ace.py`
- Протокол: [Protocol.md](Protocol.md) / [Protocol_ru.md](Protocol_ru.md)
- Примеры макросов: `ace.cfg.sample`
