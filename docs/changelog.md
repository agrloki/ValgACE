# Changelog

Все значительные изменения проекта ValgACE документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [2026-03-22] - Обновление документации Infinity Spool

### Добавлено
- Параметр `infinity_spool_debounce` - время debounce для подтверждения empty статуса (по умолчанию 2.0 сек)
- Параметр `infinity_spool_pause_on_no_sensor` - пауза при отсутствии датчика филамента (по умолчанию True)
- Описание Infinity Spool Auto-trigger системы в документации
- Автоматический мониторинг статуса активного слота во время печати

### Изменено
- Удалена запись о неработающем infinity spool из "Известные проблемы" в README
- Обновлена документация CONFIGURATION.md с новыми параметрами
- Обновлена документация COMMANDS.md с описанием auto-trigger алгоритма
- Обновлены английские версии документации

### Исправлено
- Infinity Spool Mode теперь работает корректно с auto-trigger системой

## [2025-12] - Агрессивная парковка

### Добавлено
- Параметр `aggressive_parking` - альтернативный алгоритм парковки
- Параметр `max_parking_distance` - максимальная дистанция парковки
- Параметр `parking_speed` - скорость парковки
- Параметр `extended_park_time` - дополнительное время к таймауту
- Параметр `max_parking_timeout` - максимальное время ожидания парковки

## [2025-11] - Slot Mapping

### Добавлено
- Команда `ACE_GET_SLOTMAPPING` - получить текущее отображение слотов
- Команда `ACE_SET_SLOTMAPPING` - установить отображение индекса в слот
- Команда `ACE_RESET_SLOTMAPPING` - сбросить отображение в дефолтные значения
- Команда `ACE_GET_CURRENT_INDEX` - получить текущий индекс инструмента
- Команда `ACE_SET_CURRENT_INDEX` - установить текущий индекс

---

## [2025-10] - Базовая версия

### Добавлено
- Базовое управление устройством ACE Pro
- Автоматическая смена инструмента (4 слота)
- Управление сушкой филамента
- Режим бесконечной катушки (infinity spool)
- REST API через Moonraker
- Веб-интерфейс ValgACE Dashboard
